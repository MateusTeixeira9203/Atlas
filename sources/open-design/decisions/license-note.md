# Decisão de Licença: Open Design (Apache 2.0)

**Data:** 2026-06-22
**Fase:** 2 — Integração
**Status:** DECISÃO TOMADA

---

## Situação

Open Design usa **Apache 2.0**, que exige:
1. Incluir cópia da licença junto ao trabalho derivado.
2. Indicar claramente quaisquer modificações feitas.
3. Preservar avisos de copyright do arquivo original.

---

## Decisão

Todo arquivo extraído do open-design para o `cpe-design/` deve incluir
um cabeçalho de proveniência:

```
# PROVENIÊNCIA
# Fonte: nexu-io/open-design (https://github.com/nexu-io/open-design)
# Licença: Apache 2.0
# Commit: 1cb7eae45180295d16d0c7e54c5100b56efcae66 (2026-06-22)
# Adaptações: [descrever o que foi alterado]
```

O arquivo `CREDITS.md` da raiz do CPE deve listar o open-design com link
para a licença original.

Uma cópia de `LICENSE-open-design` será mantida em `cpe-design/`.
